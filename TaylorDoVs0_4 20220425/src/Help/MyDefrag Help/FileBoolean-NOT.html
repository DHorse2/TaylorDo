FileBoolean - NOT

Logically negate (invert) a file boolean. If the boolean is true then the result is false, and if the boolean is false then the result is true.

Syntax

not ( FILEBOOLEAN )

Example

FileSelect
  not ( Name("*.zip") or Name("*.arj") )
FileActions
  ...
FileEnd
