VolumeBoolean - NOT

Logically negate (invert) a volume boolean. If the boolean is true then the result is false, and if the boolean is false then the result is true.

Syntax

not ( VOLUMEBOOLEAN )

Example

VolumeSelect
  not ( Name("c:") or Name("d:") )
VolumeActions
  ...
VolumeEnd
