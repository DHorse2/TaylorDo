FileBoolean - Archive

Select all the items that have the "archive" attribute set (yes) or not set (no). Applications use this attribute to mark files for backup or removal.

Syntax

Archive(yes)
Archive(no)

Example

FileSelect
  # Select all the items that have the "archive" attribute.
  Archive(yes)
FileActions
  ....
FileEnd
