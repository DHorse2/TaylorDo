FileBoolean - Hidden

Select all the items that have the "hidden" attribute set (yes) or not set (no). Hidden items are not included by Windows in an ordinary directory listing.

Syntax

Hidden(yes)
Hidden(no)

Example

FileSelect
  # Select all the items that have the "hidden" attribute.
  Hidden(yes)
FileActions
  ....
FileEnd
