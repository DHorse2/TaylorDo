FAQ General information - What is the "Average end-begin distance"?

When Windows has finished reading a file then the harddisk heads will have to move to the beginning of whatever file is needed next. MyDefrag calculates the average distance from the end of every file to the beginning of every other file, and shows this number in the report (see the logfile). A lower number means that the files are better packed together and can be accessed more quickly.

Note: The lowest possible average distance can be achieved by sorting by filesize.
