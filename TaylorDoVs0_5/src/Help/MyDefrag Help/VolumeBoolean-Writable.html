VolumeBoolean - Writable

Select the volume if it is Writable(yes) or not Writable(no).

Note: Windows 2000 does not have the capability to make volumes read-only, all volumes are always writable.

Syntax

Writable(yes)
Writable(no)

Example

VolumeSelect
  Writable(yes)
VolumeActions
  ...
VolumeEnd
