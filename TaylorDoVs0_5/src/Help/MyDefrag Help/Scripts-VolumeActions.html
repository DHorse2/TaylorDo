Scripts - VolumeActions

The VolumeActions keyword is part of the  VolumeSelect structure and specifies the actions to be done on the selected volumes. There are several actions to choose from, but the basic workhorse is the  FileSelect structure.

Example

VolumeSelect
  ....
VolumeActions
  ....
VolumeEnd

Actions

 DeleteJournal
 DismountVolume
 FileSelect
 MakeGap
 ReclaimNtfsReservedAreas
 Settings
