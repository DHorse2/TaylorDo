Scripts - FileActions

The FileActions keyword is part of the  FileSelect structure and specifies the actions to be performed on the selected items (files, directories). There are several actions to choose from.

Example

FileSelect
  ....
FileActions
  Defragment()
FileEnd

Actions

 AddGap
 Defragment
 FastFill
 ForcedFill
 MoveDownFill
 MoveToEndOfDisk
 MoveUpToZone
 PlaceNtfsSystemFiles
 Settings
 SortByCreationDate
 SortByImportSequence
 SortByLastAccess
 SortByLastChange
 SortByName
 SortByNewestDate
 SortBySize
