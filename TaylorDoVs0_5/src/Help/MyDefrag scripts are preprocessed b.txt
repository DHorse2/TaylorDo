MyDefrag scripts are preprocessed before being executed by the app.  Any occurrences of !include "????"! (ie. !include "..\Volume\VolumeStatisticsDetailGet.MyDc"!) will insert that file into the script. This is a recursive process. Processing then starts. I generate (make) an output (or preview) document. It keeps track of the overall line number and line number of the include to the left of any line.
